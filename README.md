# Pikachu

画一个皮卡丘，支持移动端访问

## 开发中遇到的问题

1. 关于堆叠上下文

   **问题**：因为开始给父元素设置了相对定位，触发了堆叠上下文，所以z-index:-1在背景上面，如果在父元素外面加一个div会把堆叠上下文取消，导致z-index:-1在背景之下，产生一个意外bug

   **解决**：

   - 让父元素触发堆叠上下文

   - 设置正值的z-index，高的盖住低的

   - 把舌头的位置提前，因为同一层级，后面会盖住前面

2. 调速不能用setInterval()，因为只会读取一次速度

   **解决**:使用setTimeout来代替

3. 添加 `-webkit-overflow-scrolling: touch;` 在移动端滚动更顺化
